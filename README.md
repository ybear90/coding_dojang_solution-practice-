알고리즘 풀이를 완성을 시키는 선에서 그치는 것이 아닌, 직접 풀어보고 다른 사람들의 여러가지 풀이를 비교해 본 다음 좋은 풀이는 배워서 써먹을 수 있도록
하고 상대적으로 이상하거나 좋지 않은 풀이는 이렇게 하면 좋겠다라는 아이디어를 얻거나 개선도 해볼 수 있도록 좀 더 능동적으로 알고리즘을 학습하기 위함.
